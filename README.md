### :seven: 회고록

- 남승현(main):   
1. model 생성 시 manytomany Field 사용하는 것이 익숙하지 않았다.
2. 댓글삭제 구현 시 csrf를 추가하지 않은 실수를 했다.
3. 다른 팀원과 코드를 merge할 경우 곂치는 부분을 수정하는 것이 힘들었다.
4. 팀원과의 원만한 소통을 통해 프로젝트를 완성할 수 있었다.
